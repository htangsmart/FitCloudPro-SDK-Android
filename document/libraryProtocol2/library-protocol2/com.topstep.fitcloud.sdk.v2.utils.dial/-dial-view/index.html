DialView

open class DialView : View

Used to show different dial backgrounds, styles, and positions. 1. Shape. setShape 2. Background ScaleType. setBackgroundScaleType 3. Style Position. setStylePosition 4. Dial background bitmap. setBackgroundBitmap 5. Dial style bitmap. setStyleBitmap

setCheckParams and setChecked will set a selected effect.

The background bitmap and style bitmap can be any format and size picture. But dial bin file need RGB_565 format pictures and the size must match of dial shape. Your can use createActualBackground and createActualPreview to create the actual bitmap for bin file.

Note:If you use DialView_dv_background_src And DialView_dv_style_src set default background and style bitmap. It is recommended to put related pictures into the drawable-nodpi directory, because the pictures related to the dial are designed according to specific sizes.
