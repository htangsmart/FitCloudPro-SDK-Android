setQrCode

@CheckResult
abstract fun setQrCode(type: Int, content: String): Completable

Set a QR codes display on device.

The all QR codes add define in FcQrCodeType, but the device may only support some of them.

The QR code has four groups:

  • Collection Code: If FcDeviceInfo.Feature.COLLECTION_CODE is support. This groups QR codes content limit 274 bytes

  • Business Card: If FcDeviceInfo.Feature.BUSINESS_CARD is support. This groups QR codes content limit 274 bytes

  • Nucleic Acid Code: If FcDeviceInfo.Feature.NUCLEIC_ACID_CODE is support. This groups QR codes content limit 347 bytes

  • Qr Code Extension 1 : If FcDeviceInfo.Feature.QR_CODE_EXTENSION_1 is support. This groups QR codes content limit 347 bytes

For example, if you want set FcQrCodeType.COLLECTION_CODE_PAYPAL to device:

//Check if a certain group of QR codes is supported, such as displaying the settings entry for a certain group of QR codes on the page
val isSupport = connector.configFeature().getDeviceInfo().isSupportFeature(FcDeviceInfo.Feature.COLLECTION_CODE)
if (!isSupport) return

//requestSupportQrCodes for detail judgment, determine whether [FcQrCodeType] is supported under a certain group.
val types = connector.settingsFeature().requestSupportQrCodes().onErrorReturnItem(emptyList()).blockingGet()
if (types.contains(FcQrCodeType.COLLECTION_CODE_PAYPAL)) {
    //supported
    connector.settingsFeature().setQrCode(FcQrCodeType.COLLECTION_CODE_PAYPAL, "qr codes content").onErrorComplete().subscribe()
}
