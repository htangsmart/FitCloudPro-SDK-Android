Connect with device address. When the connection is successful, login or bind the device and do some basic settings for it. The difference between login and bind is that the bind will clear the data(steps,sleep,heart rate,etc...) on the wristband, and the login will not. The connection state will callback with observerConnectionState The connection error will callback with observerConnectionError

Parameters

address

The device address

userIdentity

The user identity. When converted to bytes with getBytes, the bytes length must be less than 28. Parts longer than 28 will be ignored

bindOrLogin

True for bind, false for login.

sex

User sex info. True for man. False for woman.

age

User age

height

User height. Unit cm

weight

User weight. Unit kg
