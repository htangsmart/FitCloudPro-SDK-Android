Perform initialization work, and provide external access interface. You can directly make your own Application class inherit it, or initialize it in the onCreate method. 

public class YourApplication extends WristbandApp{
    ....
}

or

public class YourApplication extends Application{
     public void onCreate() {
         super.onCreate();
         WristbandApp.init(this);
     }
}

RxJavaPlugins.setErrorHandler(new Consumer() {
    public void accept(Throwable throwable) throws Exception {
        if (throwable instanceof UndeliverableException && throwable.getCause() instanceof BleException) {
            return; // ignore BleExceptions as they were surely delivered at least once
        }
        // add other custom handlers if needed
        throw new Exception(throwable);
    }
