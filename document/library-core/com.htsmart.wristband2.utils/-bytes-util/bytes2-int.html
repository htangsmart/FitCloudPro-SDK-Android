byte数组中取int数值

当length=4，代表src为有符号整数的字节数组 当length>0&&length<4，代表src为无符号整数的字节数组

一般适用于 int32 的字节数组，转换为int 和 unsigned int16 的字节数组，转换为int

Return

int数值

Parameters

src

byte数组

offset

从数组的第offset位开始

length

一共需转换几个字节, length>0&&length<=4

bigMode

是否是大端模式，大端模式，高字节在前面
