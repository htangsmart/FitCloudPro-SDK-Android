Contains all wristband configuration and wristband version info. WristbandVersionNotificationConfigPageConfigFunctionConfigHealthyConfigSedentaryConfigDrinkWaterConfigBloodPressureConfigTurnWristLightingConfigWarnHeartRateConfigWarnBloodPressureConfigNotDisturbConfigWomenHealthyConfig

If isExtWarnHeartRate is false, the warnHeartRateConfig will be null. If isExtWarnBloodPressure is false, the warnBloodPressureConfig will be null. If isExtNotDisturb is false, the notDisturbConfig will be null. If isWomenHealthyEnabled is false, the womenHealthyConfig will be null. If isExtProtectionReminder is false, the protectionReminderConfig will be null. If isExtHandWashingReminder is false, the handWashingReminderConfig will be null.
