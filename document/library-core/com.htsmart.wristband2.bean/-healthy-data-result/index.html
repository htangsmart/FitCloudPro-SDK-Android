The healthy real time testing result. openHealthyRealTimeDataopenHealthyRealTimeData

If a type is not turned on or the bracelet does not detect data, the corresponding value is 0.

The temperature data is a little special. In fact, the temperature may be 0℃, but due to the limitation of the bracelet sensor, it is impossible to distinguish between 0℃ and invalid data. Therefore, when the getTemperatureWrist is not equal to 0℃, it is considered to be valid data, although this will have a small probability of error. The getTemperatureBody data can only obtain valid data when the last inspection is completed, and it is 0℃ at other times.
