If isExtWarnHeartRate is true, it represents wristband support the heart rate warning function.

This setting will affect whether heart rate warning is enabled and set the warning value. Static warning refers to warning in non-exercise state. Dynamic warning refers to warning in exercise state.

When the heart rate exceeds the preset value and lasts for a certain period of time, the wristband will vibrate for warning. Generally, the threshold of dynamic warning is higher than that of static warning.
