@JvmStatic
fun calculate(dateInMillis: Long, items: List<ICalculateSleepItem>, isSorted: Boolean, isSupportSleepNap: Boolean): SleepSummary

zh:将睡眠数据划分为不同的睡眠段。 每段睡眠中的items都已经被修正，去掉了无效的状态，并且在时间上是连续的。 所有睡眠段中的items可能和传入的items差别很大，所有此方法返回的items不适合持久化，因为它不是原始的数据。

en:Divide sleep data into different sleep segments. The items in each sleep segment have been corrected, removed invalid states, and it is continuous in time. The items in all sleep segments may differ significantly from the items passed in, so the items returned by this method are not suitable for persistence, because it is not the original data.

Return

Sleep summary data, divides sleep into nighttime or daytime sleep, and divides each segment.

Parameters

dateInMillis

Which date the items belong.

items

Sleep items for a certain day.

isSorted

The items must be sorted by ICalculateSleepItem.getCalculateStartTime in ascending order. But if you have already sorted, pass true to skip the process of repeating the sorting internally.

isSupportSleepNap

Whether the items is obtained from device which support FcDeviceInfo.Feature.SLEEP_NAP.
