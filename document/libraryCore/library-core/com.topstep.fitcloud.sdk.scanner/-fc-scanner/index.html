interface FcScanner

Helper for scan "FitCloud" devices

For the FcScanner.scan process, it provides some additional implementations:

1.Filter devices. Only return FitCloud devices

Currently, we use companyId to distinguish whether it is a FitCloud device. The companyId of FitCloud devices may be one of the following: 0x5448,0x4254,0x5A54

2.Use different scanning modes for different scenes in the foreground and background.

For the foreground, use ScanSettings.SCAN_MODE_LOW_LATENCY. For the background, use ScanSettings.SCAN_MODE_LOW_POWER.

3.Optimize the exception of BleScanException.UNDOCUMENTED_SCAN_THROTTLE.
